    getSnowProperty

    Introduced in

    Version:

    VBS4 22.1.0

    Description

    Description:

    Returns snow property for surfaces, such as roads, trees, and buildings.

    To get road plowing or ice plowing (VBS4 23.2+), use the "roadPlowing" or the "icePlowing" property parameter. The return value is the amount of snow / ice removed from surfaces, in meters .

    To get snow on buildings / trees, use the "buildings" / "trees" property parameter. The returned value is a normalized value in the range 0 - 1 .

    Syntax

    Syntax:

    getSnowProperty property

    Parameters:

    • type: String - Snow property to return. Can be:

      • "roadPlowing" - Road plowing.

      • "icePlowing" - Ice plowing (VBS4 23.2.+).

      • "trees" - Snow on trees.

      • "buildings" - Snow on buildings.

    Return Value:

    Number - Amount of snow or road / ice plowing. Road / ice plowing is in meters, snow on buildings / trees is in normalized values, in the range 0 - 1.

    Examples

    Examples:

    		 
    _roadPlowing = getSnowProperty "roadPlowing";

    _icePlowing = getSnowProperty "icePlowing";

    _snowOnTrees = getSnowProperty "trees";

    _snowOnBuildings = getSnowProperty "buildings";
