    setDefaultFogOfWar


    Introduced in

    Version:

    VBS3 20.1, VBS4 19.1

    Description

    Description:

    Applies the default Fog Of War (FOW) setting, used for determining entity visibility, for systems such as AI and / or MRHD.

    Syntax

    Syntax:

    setDefaultFogOfWar [mode, observer]

    Parameters:

    • mode: String - FOW mode. Can be:

      • "ORBAT" (Default) - All entities in the line-of-sight of the player, regardless of side. And all entities within the player Higher echelon (V19.1.0+).

      • "OFF": Shows nothing.

      • "ALL": Shows all units present.

      • "SIDE": Show all units of the specified observer side(s).

      • "VISIBLE": Shows all units physically visible to specified observer .

      • "VISIBLESIDE": Show all units of the specified observer side(s), and those physically visible to them (V19.1.0+).

      • "NONE": Shows only the player. In V19.1.0+, also accounts for the new FOW system.

    • observer: Object - Observer entity.

    Return Value:

    Nothing

    Examples

    Examples:

    		 
    setDefaultFogOfWar ["SIDE", unit1];
