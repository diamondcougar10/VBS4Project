    Commands and Functions by Category

    Use this page to find the appropriate commands when you are dealing with specific scripting situations (for example, all commands that have to do with triggers, or to control unit behavior, and so on).

    To make locating commands easier, some of them may appear in more than one category.

    Note that the lists contain scripting functions, as well as scripting commands.

    Functions are typically identified by the prefix fn_, and require special syntax. See Functions A-Z for more information.
