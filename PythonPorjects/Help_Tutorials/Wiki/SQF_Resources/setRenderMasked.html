    setRenderMasked


    Introduced in

    Version:

    VBS2 2.12

    VBS4 Description (VBS4 22.2+)

    Description:

    Sets of named class used for object classification , defined using setRenderMasking.

    This command overrides what is set using the classification-rendering XML file in setRenderMasking.

    VBS4 Syntax (VBS4 22.2+)

    Syntax:

    setRenderMasked [name_class, type, enabled, color ]

    Parameters:

    • name: String - Named class, as defined using setRenderMasking.

    • type: String - Can be either "material" or "property", based on what you want to classify.

    • enabled: Boolean - Enable (true) or disable (false).

    • color: Array - RGB color (optional, can be omitted, if enabled is set to false). The color can override the color set using the classification-rendering XML file in setRenderMasking.

    Return Value:

    Nothing

    VBS4 Examples (VBS4 22.2+)

    Examples:

    		 
    setRenderMasked ["building", "property", true, [200,200,0]];
    setRenderMasked ["building", "property", false];

    VBS3 Description

    Description:

    Applies a mask, defined using setRenderMasking.

    The mask covers all elements of the specified type with a solid color. The setting is persistent between mission restarts.

    VBS3 Syntax

    Syntax:

    type setRenderMasked masked

    Parameters:

    • type: String - General type of objects that should be masked. Name is case-sensitive, and can be:

      • "ground" - Solid ground, including grass (but no bushes or rocks).

      • "sky" - Sky, including clouds.

      • "water" - Open sea, lakes and rivers.

      • "staticobj" - Any objects placed on the map: Buildings, vehicles, people, trees, rocks, and so on.

    • masked: Boolean - If true, the mask is applied.

    Return Value:

    Nothing

    VBS3 Alternative Syntax

    Syntax:

    object setRenderMasked masked

    Parameters:

    • object: Object - Specific objects that should be masked.

    • masked: Boolean - If true, the mask is applied.

    Return Value:

    Nothing

    VBS3 Examples

    Examples:

    		 
    "sky" setRenderMasked true;
    wall1 setRenderMasked true;
