    param

    Introduced in

    Version:

    VBS4 23.1.0

    Description

    Description:

    Extracts a single value with a given index from the input argument . When used without an argument, as shown in main syntax, the internal variable _this is used as the argument. If the input argument is not an array, it is converted to a 1-element array.

    If the extracted item of the input with the given index is undefined, of the wrong type, or of the wrong length (if the item is an array), a default value is used instead.

    Syntax

    Syntax:

    param [ index, defaultValue, expectedDataTypes, expectedArrayCount]

    Parameters:

    • index: Number - Index of the required item in the input array.

    • defaultValue: Anything - (Optional) Default value to return, if the input is undefined, of the wrong type, or of the wrong length (if the item is an array).

    • expectedDataTypes: Array of Data Types - (Optional) Checks if the passed value is of one of the listed Data Types. If not, a default value is used instead. An empty array [] means that every data type is accepted.

    • expectedArrayCount: Number or Array - (Optional) Single size or array of sizes. If the passed input value is an array, checks that it has a certain number of elements. If not, a default value is used instead. An empty array [] means that any size is accepted.

    Return Value:

    Anything - Returns the extracted value on success, or the default value otherwise. Returns Nothing, if a syntax error occurrs.

    Alternative Syntax

    Syntax:

    argument param [index, defaultValue, expectedDataTypes, expectedArrayCount]

    Parameters:

    • argument: Anything - A rray of parameters is expected. If a non-array value is passed, it is converted to a 1-element array.

    • index: Number - Index of the required item in the input array.

    • defaultValue: Anything - (Optional) Default value to return, if the input is undefined, of the wrong type, or of the wrong length (if the item is an array).

    • expectedDataTypes: Array of Data Types - (Optional) Checks if the passed value is of one of the listed Data Types. If not, a default value is used instead. An empty array [] means that every data type is accepted.

    • expectedArrayCount: Number or Array - (Optional) Single size or array of sizes. If the passed input value is an array, checks that it has a certain number of elements. If not, a default value is used instead. An empty array [] means that any size is accepted.

    Return Value:

    Anything - Returns the extracted value on success, or the default value otherwise. Returns Nothing, if a syntax error occurrs.

    Examples

    Example 1:
    [1, 2, 3] call
    {
    private _one = param [0, 1];
    private _two = param [1, 2];
    private _three = param [2, 3];
    // ...
    };

    Example 2:
    [123] call
    {
    private _val = param [0];
    };
     
    // The following code produces the same result as the preceding one
    123 call
    {
    private _val = param [0];
    };

    Example 3:
    _z = position player param [2, 0];
    if (_z > 10) then
    {
    hint "YOU ARE FLYING!";
    };

    Example 4:
    fnc =
    {
    private _pos = param [0, [0,0,0], [objNull, []], [2,3]];
    private _rad = param [1, 0, [0]];
    _pos nearObjects _rad;
    };
     
    [position player, 25] call fnc; // ok
    [player, 25] call fnc; // ok
    [25, player] call fnc; // default values are used
