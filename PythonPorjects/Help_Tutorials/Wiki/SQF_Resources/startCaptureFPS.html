    startCaptureFPS



    Introduced in

    Version:

    3.7

    Description

    Description:

    Starts FPS logging at the specified interval.

    A file with the logged measurements is created when the stopCaptureFPS command is executed.

    Syntax

    Syntax:

    startCaptureFPS interval

    Parameters:

    • interval: Number - Time (in milliseconds) between measurements

    Return Value:

    Nothing

    Examples

    Examples:

    		 
    Log performance every 200 milliseconds, and write the output to c:\bi\fpslog.txt:
    // an fps measurement will be logged every 200ms
    startCaptureFPS 200; 

    // action to be measured
    for "_i" from 0 to 100 do {
      systemChat str _i;
      sleep .01;
    };

    // log file is created
    stopCaptureFPS "c:\bi\fpslog.txt";
