    • Description

    • Class properties

    • Icons

    Description

    Class names, descriptions and images of standard markers.

    The class name (left entries in the list) is what has to be used in the createMarker command.

    Class properties

    Example: class CfgMarkers { class Flag { name = "Marker description"; icon = "\ca\ui\data\markers\marker1.paa"; color[] = {1, 0, 0, 1}; size = 32; shadow = true; scope = private; };
